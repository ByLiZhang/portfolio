PartPig

Trading of used car parts made easy

    PartPig is a web-based e-Commerce platform built to facilitate the trading of used car parts. It is designed to efficiently connect the buyer and seller by focusing on user interface and experience (UI/UX). Once signed in, a seller can conveniently upload images, draft and edit part information, and then post the parts for sale within minutes. Using the search bar and filters, a buyer can easily search for the parts he/she is interested in, and then add the found items into the shopping cart for checkout.

    PartPig is currently in its Beta version, and will be deployed for real use in the near future.

Technologies used

React PHP MySQL AWS (EC2, S3) SendGrid Transactional Emails Confluence/Jira

TrailMix

A trail-finder for all hikers

    TrailMix is a trail-finding web application. Its prototype was originally created during a two-day hackathon using JavaScript and jQuery, and was later rebuilt with ReactJS and Redux.

    TrailMix features an easy-to-use interface. When a location is specified by the user, a list of hiking trails is populated and displayed on the map using data obtained through REI and Google Maps APIs. Further information such as trail details, driving directions and weather forcast are also only a few clicks away.

Technologies used

React Redux CSS3 Google Maps APIs REI API Dark Sky Weather API

Live Site GitHub

Nature Connect

Enjoy a game with beautiful scenes

    NatureConnect is a web application version of the classic Connect-Four game. It features an appealing nature-themed interface, and emphasizes on smooth and realistic game play. It uses JavaScript for game logic, jQuery for DOM manipulation, and jQuery-UI for token animation. When in single-player mode, an AI is provided as the opponent.

Technologies used

JavaScript jQuery jQuery-UI HTML5 CSS3
